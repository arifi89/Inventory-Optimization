# Inventory Optimization Project

## Project Overview
Comprehensive inventory analysis and optimization case study for procurement strategy development.

## Project Structure
- **data/**: All data files (raw, processed, segmentation)
- **notebooks/**: Jupyter notebooks for analysis
- **src/**: Python source code modules
- **procurement_strategy/**: Strategic documentation
- **powerbi/**: Power BI dashboards and screenshots
- **reports/**: Final reports and visualizations

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Follow notebooks in sequence (01-08)
3. Review procurement strategy documents

## Key Analyses
- ABC-XYZ Segmentation
- Kraljic Matrix
- EOQ, ROP, Safety Stock Optimization
- Supplier Segmentation
- Before/After Scenarios
